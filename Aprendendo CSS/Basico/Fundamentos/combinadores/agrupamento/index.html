Agrupamento de Seletores

Pordemos aplicar as mesmas regras CSS para varios seletores diferentes.Para isso,Podemos separar os seletores utilizando uma virgula(,).

Assim,não precisamos ficar copiando e colcando os mesmos codigos para todos,nós podemos fazer um só bloco de CSS e reultilizar as mesmas regras!
