Pseudo-elemento

Usado para adicionar elemento extras na pagina sem a nescessidade de declara-lo no HTML ou seja para seleciocar uma parte dde algum elemento

Identifica-lo por ::palavra-chave

Pseudo-elemento - ::Fist-letter

Seleciona a primeira letra da primeira linha

Propiedades disponiveis

  • color
  • backgroud
  • margin
  • font
  • border
  • padding
  • propiedades de texto

Pseudo-Elemento Marker

Permite estilizar os tópicos ou númertos de uma lista (li) ou então a flecha de um elemento Sumary

Propiedades Aceitas
  • Color
  • Content
  • White-space
  • Font
  • Animation
  • Transition

Com este Pseudo-elemento Podemos alterar o texto que fica no input
